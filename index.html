Digital Safe System Project

A fully logic-based electronic safe designed and built without any microcontroller. The system operates using shift registers, flip-flops, logic gates, timers, and display modules to securely validate and process user input.In addition, an Arduino-based version was also implemented to demonstrate a programmable approach to the same functionality.
